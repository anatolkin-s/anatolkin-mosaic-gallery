# Mosaic Gallery for TYPO3

Masonry-style image gallery powered by FAL, with optional lightbox.
TYPO3 v13 LTS, PHP 8.1-8.3.

## Features
- Reads images from any FAL folder
- Masonry-like layout, responsive
- Optional Lightbox (overlay color & opacity)
- Simple design options (gutter, border color, background color)

## Installation

### Composer (recommended)
```bash
composer req anatolkin/mosaic-gallery
